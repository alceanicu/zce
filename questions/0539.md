- [ ] A)
```php
function myFunction() {
  //
}
```

- [ ] B)
```php
create myFunction() {
  //
}
```

- [ ] C)
```php
define new myFunction() {
  //
}
```

- [ ] D)
```php
func myFunction() {
  //
}
```
