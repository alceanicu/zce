Consider the following PHP script:
```php
function b($a = 4) {
    $a = $a / 2;
    return $a;
}

$a = 10;
b($a);
echo $a;
```
What will be the output?

- [ ] A) 10
- [ ] B) 5
- [ ] C) 2
- [ ] D) The script will throw an error message.

<details><summary><b>Answer</b></summary>
<p>
  Answer: <strong>A</strong>
</p>
</details>
