Consider the following PHP code, which defines an associative array of fruits and vegetables.

```php
$fruitAndVeg = array(
        'c' => 'Carrot',
        'p' => 'Tomato',
        'b' => 'Banana',
        't' => 'Potato',
        'a' => 'Apple'
);

/** line **/

$keys = array_keys($fruitAndVeg);
echo $keys[0];
```
What line of code should be substituted with /** line **/ to achieve an output of a?

- [x] A
```php
ksort($fruitAndVeg);
```

- [ ] B
```php
usort($fruitAndVeg);
```

- [ ] C
```php
sort($fruitAndVeg);
```

- [ ] D
```php
keysort($fruitAndVeg);
```
