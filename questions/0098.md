```php
$a = 'somevalue';

// ????
echo "\$a = $a; \$b = $b; \$c = $c;";
```
What should you write at line number 3 to get the following output?
$a = One; $b =Two; $c = $Three;


```php
implode($array);
```

```php

```
