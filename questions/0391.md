The PHP function array_reduce() is used to turn an array into a single value using a custom callback. What is the output of the following script?

```php
function reducer($total, $elt) {
    return $elt + $total;
}
$arr = [1, 2, 3, 4, 5];
echo array_reduce($arr, 'reducer', 1);
```

- [x] 16
- [ ] 14
- [ ] 0
- [ ] 15
