You run the following PHP script:

```php
for($x = 1; $x <= 2; $x++) {
      for($y = 1; $y <= 3; $y++) {
         if ($x == $y) continue; 
         print("x = $x  y =  $y");
      }
}

```
What will be the output? Each correct answer represents a complete solution. Choose all that apply.

- [ ] A) x = 2 y = 3
- [ ] B) x = 2 y = 2
- [ ] C) x = 2 y = 1
- [ ] D) x = 1 y = 3

<details><summary><b>Answer</b></summary>
<p>
  Answer: <strong>A, C, D</strong>
</p>
</details>
