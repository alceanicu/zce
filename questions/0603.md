Which of the following is the best way to split a string on the "-=-" pattern?
```php
$string = 'apple-=-banana-=-orange';
```

- [x] A
```php
explode("-=-", $string);
```

- [ ] They all are equally proper methods
- [ ] C
```php
str_split($string, strpos($string, "-=-"))
```

- [ ] D
```php
preg_split("-=-", $string);
```
