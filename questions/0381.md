How do you create an array called $arr consisting of a single element with the value 15?

- [ ] A)
```php
$arr = array(15);
```

- [ ] B)
```php
$arr = array_shift(15);
```

- [ ] C)
```php
$arr = new array(15);
```

- [ ] D)
```php
$arr = array_push(15);
```


<details><summary><b>Answer</b></summary>
<p>
  Answer: <strong>A</strong>
</p>
</details>
