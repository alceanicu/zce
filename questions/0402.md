The str_pad() function is used pad a string to a given length using another string. What is the output of the following PHP script?


```php
$number = 5;
$len    = 3;
$pad    = '0';
echo str_pad($number, $len, $pad, STR_PAD_LEFT);
```

- [x] 005
- [ ] 500
- [ ] 000
- [ ] 30000
