Which of the following functions will you use to read a file having single line irrespective of length? Each correct answer represents a complete solution. Choose two.

- [x] A
```php
fgetss($fp);
```

- [x] B
```php
fgets($fp);
```

- [ ] C
```php
fread($fp, 1024);
```

- [ ] D
```php
fread($fp);
```
