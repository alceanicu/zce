How do you remove an element with the key 0 from the array $numbers?



- [ ] A)
```php
unset($numbers[0]);
```

- [ ] B)
```php
array_shift($numbers);
```

- [ ] C)
```php
$numbers[0] = null;
```

- [ ] D)
```php
array_pop($numbers);
```

<p>
  Answer: <strong>A</strong>
</p>
