What will be the output of the following PHP script:

```php
function modifyArray(&$array) {
    foreach ($array as &$value) {
        $value = $value + 2;
    }
    $value = $value + 3;
}

$array = array(1, 2, 3);
modifyArray($array);
print_r($array);
```
