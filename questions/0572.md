How do you declare the return type of a function?

- [x] A
```php
function sum($a, $b) :int {return $a + $b;}
```

- [ ] B
```php
function sum($a, $b) {return (int)$a + $b;}
```

- [ ] C
```php
function sum($a, $b) => int {return $a + $b;}
```

- [ ] It is not possible to define a return type
