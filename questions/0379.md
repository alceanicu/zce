In the following code, what are the values required in $a, $b, $c and $d to output 40?

```php
$values = array(
    array(
        1 => 10,
        20,
        array(30, array(40))
    ),
    array(
        2 => 50,
        array(
            array(1 => 60, 0 => 70)
        )
    )
);

echo $values[$a][$b][$c][$d];
```

- [ ] A) $a = 0, $b = 3, $c = 1, $d = 0
- [ ] B) $a = 0, $b = 1, $c = 1, $d = 0
- [ ] C) $a = 0, $b = 1, $c = 0, $d = 0
- [ ] D) $a = 1, $b = 3, $c = 1, $d = 0

<p>
  Answer: <strong>A</strong>
