How do you declare a function to accept a variable (unknown) number of arguments in PHP 7?

- [ ] A)
```php
function foo(...$parms) {}
```

- [ ] B)
```php
function foo($parms <multi>) {}
```

- [ ] C)
```php
function foo(<multi> $parms) {}
```

- [ ] D)
```php
function foo(@multi $parms) {}
```


<details><summary><b>Answer</b></summary>
<p>
  Answer: <strong>A</strong>
</p>
</details>
