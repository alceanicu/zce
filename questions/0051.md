Which of the following options shows the correct IF statement format?

- [ ] A)if (expression) { 
    // code to execute if the expression evaluates to true
}
- [ ] B)if {expression} (
    // code to execute if the expression evaluates to true 
) fi
- [ ] C)if {expression} ( 
    // code to execute if the expression evaluates to true
)
- [ ] D)if (expression) { 
    // code to execute if the expression evaluates to true 
} fi

<details><summary><b>Answer</b></summary>
<p>
  Answer: <strong>A</strong>
</p>
