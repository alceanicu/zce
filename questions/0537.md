Which of the following comparisons will evaluate to true?

```php
var_dump(
    't' == t, // Notice
    1 === "1time",
    "top" == 0,
    "top" === 0,
    1 == "1time"
);
```

- [ ] A) 't' == t
- [ ] B) "top" == 0
- [ ] C) 1 == "1time"
- [ ] D) 1 === "1time"

<details><summary><b>Answer</b></summary>
<p>
  Answer: <strong>A, B, C</strong>
</p>
</details>
