```php
<html>
  <head>
    <title>PHP</title>
  </head>
  <body>
   <?php echo 'This is some sample text'; ?>
  </body>
</html>
```
Which of the following tags is used in the php script?

- [ ] A) Standard tag
- [ ] B) ASP tag
- [ ] C) Script tag
- [ ] D) Short tag

  Answer: <strong>A</strong>
