Which is a valid way for calling test function?
```php
function test(?int $n) {
    var_dump($n);
}
```

- [ ] A)
```php
test(null);
```

- [ ] B)
```php
test('1');
```

- [ ] C)
```php
test(1);
```

- [ ] D)
```php
test();
```


<details><summary><b>Answer</b></summary>
<p>
  Answer: <strong>A, B, C</strong>
</p>
</details>
