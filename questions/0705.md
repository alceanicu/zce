Considering the following code which of the statements below is true ?

```php
class entity {
    public $name;
}
$human = new entity();
$dog = new entity();
$human->name = 0;
$dog->name = "";
```

- [x] A
```php
($human == $dog)
```

- [x] B
```php
($human->name == $dog->name)
```

- [ ] C
```php
($human === $dog)
```

- [ ] D
```php
($human->name instanceof $dog->name)
```
