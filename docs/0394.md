Remembering that keys are not reset when using natsort(), what is the output of the following PHP script?

```php
$filenames = array(
    'img12.png',
    'img7.png',
    'img21.png',
    'img1.png'
);

natsort($filenames);
$values = array_values($filenames);
echo $values[1];
```
Enter the exact script output

- [x] img7.png
- [ ] img12.png
- [ ] img121.png
- [ ] img1.png
