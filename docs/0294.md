Consider the following PHP script:

```php
 $fp = fopen('file.txt', 'r');
 $string1 = fgets($fp, 512);
 fseek($fp, 0);
```
Which of the following functions will give the same output as that given by the fseek() function in the above script?

- [x] A
```php
rewind()
```

- [ ] B
```php
fgets()
```

- [ ] C
```php
fgetss()
```

- [ ] D
```php
file()
```
