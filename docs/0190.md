Which of the following functions will output the current time in the 08:26 am format? Each correct answer represents a complete solution. Choose all that apply.

- [x] A
```php
print date('H:i a');
```

- [x] B
```php
print strftime('%I:%M %p');
```

- [ ] C
```php
print date('G:i a');
```

- [ ] D
```php
print date('Y:i a');
```
